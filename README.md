# Scientific Archive on Neurodiversity

This repository curates peer-reviewed scientific literature on:

- Autism Spectrum Condition
- ADHD
- Migraine and headache disorders
- Genetic and neurobiological mechanisms
- Functional and psychiatric comorbidities

The project integrates perspectives from neuroscience, genetics,
bioinformatics, and lived experience.

## Purpose

- Academic reference
- Research portfolio
- Educational resource
